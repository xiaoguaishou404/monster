## Intro

这是我在学习 Vue3 时制作的一个 UI 组件库 👀

这款组件库其实是我为了总结这段时间自己的学习成果而写的，全程亲手编写，尽量不采用第三方库

**Therefore**: 强烈不建议你将此 UI 库用于生产环境。但如果你是抱着看源代码的目的来的，那么这个库还是值得一看，也欢迎点评



## Install

`yarn add monster-vue-ui`

or

`npm install monster-vue-ui `
